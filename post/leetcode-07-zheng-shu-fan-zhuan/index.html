Leetcode07 - 整数反转

2021-11-14

参考：
Python中[:]与[::]的用法

class Solution:
    def reverse(self, x: int) -> int:
        if x==0:
            return 0
        str_x = str(x)
        x = ''
        if str_x[0]=='-':
            x += '-'
        x += str_x[len(str_x)-1::-1].lstrip("0").rstrip("-")
        x = int(x)
        if -2**31<x<2**31-1:
            return x
        return 0
